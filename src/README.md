## Team Members

## Project Description
Implementation of a content-based image retrieval (CBIR) system using multiple feature extraction methods:
- Baseline matching (7x7 center square)
- Color histogram matching
- Multi-histogram matching with spatial layout
- Texture and color features
- Deep network embeddings (ResNet18)

## Setup

### Prerequisites
- C++11 compiler (clang++)
- OpenCV 4.x
- macOS with Homebrew

### Build
```bash
make all
```

### Clean
```bash
make clean
```
